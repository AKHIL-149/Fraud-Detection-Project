## What Users Need to Do After Cloning

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Download dataset:**
   - See DATA_SETUP.md for instructions
   - Download from Kaggle: https://www.kaggle.com/datasets/ealtman2019/credit-card-transactions
   - Place files in detection_data/ directory

3. **Run the pipeline:**
   ```bash
   python data_analysis.py          # EDA and visualization
   python feature_engineering.py    # Create 67 features
   python model_training.py         # Train 4 models
   ```

4. **Generated files:**
   - detection_data/transactions_with_features.csv
   - models/*.pkl (6 model files)
   - Additional plots in plots/
